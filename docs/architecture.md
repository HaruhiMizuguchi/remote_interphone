# システム全体構成

## 信号フロー

```
スマホ
  ↓
インターネット / Wi-Fi
  ↓
Pico WH
  ↓
PWM信号
  ↓
サーボモータ
  ↓
インターホンの開錠ボタンを物理的に押す
```

## 構成要素

| 要素 | 役割 |
|------|------|
| スマホ | 操作指示を送る |
| Wi-Fi | Pico WH への通信経路 |
| Pico WH | サーボへの PWM 制御 |
| サーボモータ | ボタンを物理的に押す |
| 5V ACアダプタ | Pico WH とサーボの電源供給 |

## 電源構成（概要）

Pico 経由でサーボに給電すると電流不足・電圧降下で不安定になるため、
**Pico WH とサーボを 5V 電源に並列接続する**設計を採用。

```
5V電源
  ├─ Pico WH
  └─ サーボモータ
```

詳細は [power-design.md](power-design.md) を参照。

## 設計のポイント

1. サーボ電源は Pico から取らない
2. Pico とサーボは 5V 電源に並列接続
3. GND は必ず共通にする
4. 信号線は Pico の GPIO 1本だけ
5. まずは外部連携より先に、ボタンを確実に押せる機械部分の調整を優先する
