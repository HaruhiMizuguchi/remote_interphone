# 物理配線

## サーボモータの配線

| ワイヤ色 | 接続先 |
|----------|--------|
| 赤 | +5V（ACアダプタから直接） |
| 黒 / 茶 | GND（共通） |
| 黄 / 橙 / 白 | Pico WH の GPIO（PWM 出力用） |

## Pico WH の配線

| ピン | 接続先 |
|------|--------|
| VSYS | +5V（ACアダプタから直接） |
| GND | GND（サーボと共通） |
| GPIO（PWM対応ピン） | サーボ信号線 |

## 配線図

```
DCジャック-ネジ端子変換
  │
  ├─ +5V ───┬──→ Pico WH VSYS
  │         └──→ サーボ 赤線
  │
  └─ GND ───┬──→ Pico WH GND
             └──→ サーボ 黒/茶線

Pico WH GPIO ──→ サーボ 信号線（黄/橙/白）
```

## 物理的な接続方法

USB ネジ端子アダプタが無いため、以下の構成で接続する。

```
5V ACアダプタ
  ↓
DCジャック → ネジ端子変換アダプタ
  ↓
ブレッドボード or ジャンパワイヤで分岐
  ├─ Pico WH
  └─ サーボモータ
```

## 注意事項

- GND は必ず Pico WH とサーボで共通にすること（別 GND だとPWM信号が正しく伝わらない）
- サーボの電源を Pico の VBUS や 3V3 から取らないこと
- GPIO ピンの選定は PWM 対応ピンであること（Pico はほぼ全 GPIO が PWM 対応）
